Trafficlock

Ever been late to work due to traffic? We can help.

Download this project as a .zip file Download this project as a tar.gz file

A smart alarm clock that wakes you up in time. The app analyzes current traffic times using ArcGIS and weather trends using World Weather Online to wake up the user in time to get to their destination. After configuring your settings, all that needs to be done is to press the "Activate" button for the app to wake you up on time!

This app won Best Demonstration at HACKOHI/O, and was in the Top 10.
